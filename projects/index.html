CS425NetworksTCP

Computer Networks

As a part of the Computer Networks (CS425) course, we built a series of hands-on projects to deepen our understanding of network protocols, concurrency, and socket programming. We built a multi-threaded TCP chat server in C++ that featured user authentication, broadcast capabilities, and direct messaging, utilizing mutexes to ensure thread safety. We built custom iterative and recursive DNS resolvers that performed root, TLD, and authoritative queries for domain-to-IP mapping. Additionally, we implemented a raw-socket TCP client in C++ to execute the TCP 3-way handshake, handling SYN, SYN-ACK, and ACK states directly at the packet level.

Fun Projects

AIPythonRL

Chess Bot

A chess Bot made in python with the help of PyGame. This chess bot uses Min-Max search algorithm with alpha beta pruning and iterative deepening to find the best move to play against its opponent. The chess board supports moves like pawn-promotion, double pawn push, castling on king and queen side, and en passant from black's side.

It also uses a large data set of 1,80,000+ games played by GMs so that it can make the optimal opening moves.
