Adversarially Robust DeepFake Detection

Jan – Apr 2025 IIT Kanpur GitHub ↗
[ Hero Image — CNN feature maps and adversarial perturbation visualization ]

Overview

As generative models improve, distinguishing synthetic media from authentic content becomes a critical challenge. This project investigated not just detection accuracy, but adversarial robustness — how well deepfake detectors hold up when an adversary deliberately crafts inputs to fool them.

We built and evaluated multiple CNN baselines, implemented adversarial attacks, and ultimately designed a hybrid CNN+Transformer architecture that achieves strong performance under both clean and adversarial conditions.

Motivation

Most deepfake detectors are evaluated only on clean data. In practice, an adversary generating deepfakes can also optimize them to bypass detection. We wanted to answer: which architectures degrade gracefully under adversarial pressure, and can we design detectors that are inherently more robust?

Technical Details

Baseline Models

We trained four CNN architectures from scratch on CIFAKE and DFRI datasets:

  • VGG19 — deep feature extraction baseline
  • ResNet50 — residual learning for gradient stability
  • MobileNetV1 — lightweight, deployment-oriented
  • EfficientNet-B0 — compound scaling across depth/width/resolution

All baselines achieved up to 94% clean accuracy but showed significant vulnerability to adversarial perturbations.

Adversarial Attack Suite

We implemented three attack families in PyTorch:

  • FGSM (Fast Gradient Sign Method) — single-step, ε-bounded perturbations
  • PGD (Projected Gradient Descent) — iterative, stronger attack
  • One-Pixel Attack — extreme sparsity constraint, differential evolution-based

Hybrid CNN + Transformer Architecture

We designed a DeiT-based Vision Transformer combined with CNN feature extractors. The key insight: CNNs capture local texture artifacts (compression, blending boundaries), while Transformers model global semantic consistency. The hybrid leverages both signal types.

Lessons Learned

  • Clean accuracy is a misleading metric in isolation — adversarial evaluation is essential for any safety-critical vision system
  • Transformer attention mechanisms provide a natural form of robustness by attending to global structure rather than local pixel patterns
  • One-Pixel attacks, despite their extreme sparsity, expose unexpected vulnerabilities in architectures that overfit to texture features

Future Work

  • Extending to video-level deepfake detection with temporal consistency checks
  • Adversarial training as a defense mechanism — measuring the accuracy-robustness tradeoff
  • Evaluating on more recent generative model outputs (Stable Diffusion, DALL-E 3)
